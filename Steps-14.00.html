Creating All Measures Table with Measures 14

1.  In Home Tab, select Modelling and Create New table
2.  Name the table ‘All Measures’ Table
3.  Click Load
4.  Right Click on 'All Measures' and select New Measure
5.  Copy and paste the DAX formula for Total Shootings Measure on the Dax Bar 
6.  Click on the Right mark
7.  Repeat step 4-6 until you create all measures using the Dax formulas below

Measures

1.   Total Shootings = COUNT('Mass Shootings'[Incident ID])
            [Format-Whole number with 0 places and use comma(,)] 

2.  Fatalities = SUM(‘Mass Shootings’[Fatalities])
            [Format-Whole number with 0 places and use comma(,)]

3.  Shootings per Day = DIVIDE([Total Shootings], DISTINCTCOUNT('Calendar'[Date].[Day]))

4.  Shootings LY = CALCULATE('All Measures'[Total                       Shootings],DATEADD('Calendar'[Date],-1,YEAR))

5.  Fatalities per Day = DIVIDE([Fatalities], DISTINCTCOUNT('Calendar'[Date].[Day]))*-1

6.  Fatalities LY = CALCULATE(SUM(‘Mass Shootings’[Fatalities]),DATEADD('Calendar'[Date],-1,YEAR))

7.  Fatalities per Year = SUM('Mass Shootings'[Fatalities]) /  DISTINCTCOUNT('Calendar'[Year])

8.  Injuries = SUM('Mass Shootings'[Injuries])

9.  Injuries LY = CALCULATE(SUM('Mass Shootings'[Injuries]),DATEADD('Calendar'[Date],-1,YEAR))

10.  Shootings YoY% = DIVIDE([Total Shootings]-[Shootings LY],[Shootings LY],0)
                  [Format-percentage with 2 decimal places]

11. Fatalities YOY% = Fatalities YOY% = DIVIDE([Fatalities]-[Fatalities LY],[Fatalities LY],0)
                   [Format-percentage with 2 decimal places]

12. Injuries YoY% = DIVIDE([Injuries]-[Injuries LY],[Injuries LY],0)
                   [Format-percentage with 2 decimal places]
 
13.  Arrow YOY Shootings = Switch (True (), [Shootings YoY%] > 0, UNICHAR (129033), UNICHAR (129035))

14.  Arrow Fatalities YOY% = Switch (True (), [Fatalities YoY%] > 0, UNICHAR (129033), UNICHAR (129035))

15.  Arrow Injuries YOY% = Switch (True (), [Injuries YoY%] > 0, UNICHAR (129033), UNICHAR (129035))
