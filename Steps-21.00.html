Shootings and Fatalities per Day Chart 21

1.  Click on an empty space on the canvas, and from the Visualizations pane select Line and Stacked         Column Chart.
2.  Drag Date, Year and Month Year column from Calendar table to the X- axis. Followed                         by Fatalities per Day and Shootings per Day measures on Column Y-axis, then drag the                     following measures into Line Y-axis (Fatalities YoY%, Injuries YoY% and Shootings YoY %) 
3.  Drag all these measures into the Tooltips field: (Fatalities YoY%, Injuries YoY%, Shootings YoY%,             Fatalities, Injuries and Fatalities per Day)
4.  In the Formatting Tab->X-axis: On, Font= Segoe UI Bold, size= 9, Color = Black. Title: Off. 
     Y-axis: On, Font = Segoe UI Semibold, Click on Bold, size= 9, Color= Black, Title: Off.
     Secondary y-axis: On, Align Zeros = On, Font= Segoe UI Semibold, Click on Bold, Size = 9, Color =       Black, Title: Off. 
     Legend: On, Position = Top left, Text: Font= Segoe UI Semibold, Size= 9, Color = Black, Title: On.
    Columns: Fatalities per Day, Color #AD5129, Shootings per Day #78191F.
    Line: Shape= Solid, Colors:  Shootings YoY% = #E66C37, Injuries YoY% #6B007B, Fatalities YoY%          #0D599E, Shade area: On. 
    Data labels: On, series=Shootings Per Day show Data labels = On, Position line = Auto, Label                density = 50%, Values: Font=Segoe UI, Size = 9, Color #6C6966, Background: On, Color = White,          Transparency=30%.  Data labels for all other lines turned off.
5.  In the Formatting Tab->General->Properties->Size: Height = 298, Width = 537, Position:                       Horizontal = 203, Vertical = 92, Title: Shootings and Fatalities per Day, Font = Segoe UI Bold,                 Size = 12, Color = White, Background: Black, Center, Effect: Background: On, Color                                 #E6E6E6, Transparency=67%. 
     Tooltips: On, Text Font = Segoe UI, Size = 9, Label Color#C8C8C8, Value Color #EAEAEA and                 Background: Color = Black, Transparency = 5%
