Creating Total Shootings by City or County Map 24

1.  Click on Visualizations and choose Map. On Location add City or County column                                 from ‘Mass Shootings’ Table. Drag Total Shootings from 'All Measures' Table to the Bubble             Size. 
2.  a. In the Formatting Tab->Visual->Map Settings->Style = Grayscale, Show labels = Off.    Controls-       >Zoom buttons, Geocoding culture = Auto, Bubbles->Size = 0px. Category labels = 0, Values-             >Font = Segoe UI, Size = 9, Color #000000. Enable Conditional Formatting-> Format style =                 Gradient, Field = Total Shootings, Minimum #C06E6E, Center #F0A787, Maximum #81040E.
3.  b. In the Formatting Tab->General->Properties->Size: Height=327, Width=527, Position,                       Horizontal   =753, Vertical = 393. Title = On, Text=Total Shootings by City or County. Font =                 Segoe UI Bold, Size = 12, Text Color #FFFFFF, Background Color #000000. Alignment = Center.
4.  Take screenshot of the Gradient and save it in your device so that you can use it as a Legend for           the Map.
4.  a. In the Formatting Tab->Value->Scaling = Normal.
     b. Formatting->General->Properties->Size: Height = 49, Width = 83, Position-> Horizontal =1157,       Vertical = 667. 
5.  a. To create the Map Legend. In the report view, Select Insert ribbon->Text box - >Type                         ‘Low and High’, Highlight the Text and adjust Size = 8, Font = Segoe (Bold), Click on bold.
6.  b. In the Formatting Tab->General->Properties->Size: Height = 49, Width = 152, Position->                   Horizontal =1128, Vertical = 667. Title=Off, Effects: Background: Color #FFFFFF, Transparency =             90%.
7.  In the report view, Select Insert ribbon>Image > and select the Gradient Image you created earlier       at no. 4. Select the Gradient image the text and the Map by holding shift key then go to Format           tab select Group.
