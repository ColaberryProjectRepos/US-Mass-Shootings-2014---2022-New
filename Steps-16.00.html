     a.    In the Formatting Tab->General-> Properties->Size: Height = 65, Width = 210, Position: Horizontal = 0, Vertical = 0, Background: Off. 
3.   Repeat the same step 2 for the Gun Violence Logo 
      a.  In the Formatting Tab->General-> Properties->Size: Height = 97, Width = 144, Position: Horizontal = 1136, Vertical = 0, Background: Off.
