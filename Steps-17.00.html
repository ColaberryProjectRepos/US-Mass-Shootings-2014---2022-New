Adding Date Slicer 17

1.  In the Report View, open the Visualization pane, then Select Slicer.
2.  In the fields, open ‘Calendar’ Table, drag the Date.
    a. In the Formatting Tab->Visual->Values: Font = Segoe UI Bold, Size = 8, Slider: On
    b. In the Formatting Tab->General->Properties->Size: Height = 110, Width = 203, Position: Horizontal = 3, Vertical = 65, Title: Date, Font = Segoe UI Bold, Size =15, Horizontal alignment= Center.
