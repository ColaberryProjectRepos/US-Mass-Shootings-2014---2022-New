Adding Sub-KPI Cards 19

1.   Click on Card in Visualizations, drag Shootings Per Day to fields value box, 
     a.  In the format-> Formatting Tab->Visual->Callout value: Font = Segoe UI Bold, Size =10, Color         #78191F, Category label = Off
     b.  In the format->General->Properties->Size: Height =40, Width = 67, Position: Horizontal = 0,           Vertical = 252, Effect: Background: On, Color = White, Transparency= 68%. Title->On, Title text =           per Day, Font = DIN, Size = 8, Color #78191F, Alignment = Center.
2.  For the 2nd Sub KPI copy and paste the 1st Sub KPI and drag Fatalities Per Day to the fields value       box
    a.   In the format-> Formatting Tab->Visual->Callout value: Font = Segoe UI Bold, Size =10, Color        #AD5129, Category label= Off
    b.   In the format->General->Properties->Size: Height =40, Width = 70, Position: Horizontal = 0,          Vertical = 669, Effect: Background: On, Color = White, Transparency = 100%. Title text = per Day,          Font = DIN, Size =8, color #AD5129
