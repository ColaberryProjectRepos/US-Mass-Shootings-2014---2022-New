Creating Fatalities VS Injuries Chart 23

1.  Click on an empty space on the Canvas and drag the Scatter chart from the                                           Visualizations. Drag State to the Values, Shootings YoY% to the X-axis, Fatalities YoY% to the Y-         axis and Injuries YoY% to the Size. 
2.  a. In the Formatting Tab->Visual: X-axis, Font = Segoe UI Semibold, Size=9, click on bold, Color          #000000, Title=On. Y-axis, Font = Segoe UI Semibold, Size=9, click on bold, Color #000000,                Title=On. Legend=On, Text Font = Segoe UI Semibold, Size=11, Color #A8337D, Title = On.                  Category label=On, Font=Segoe UI Semibold, Size=10, Color #000000, Background = Off.

    b. In the Formatting Tab->General->Properties->Size: Height=327, Width=546, Position,                        Horizontal=207, Vertical=393. Title=On, Text = Fatalities VS Injuries. Font=Segoe UI Bold, Size=12,      Text Color #FFFFFF, Background Color #000000, Alignment=Center. Effect->Background Color              #E6E6E6, Transparency 65%. Tool tips- > On, Text=Segoe UI, Size=9, Label Color #C8C8C8,                    Value Color #EAEAEA, Background Color #000000, Transparency 5%.
